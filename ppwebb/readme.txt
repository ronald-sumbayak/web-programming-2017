- buat dir baru di localhost 'ppwebb'
- copy file ke situ
- buka localhost/ppwebb di browser

1. write file
   - buka write.php
   - ketik sesuatu
   - klik save
   - liat filenya di C/xampp/htdocs/ppwebb/note.txt

2. read and write file
   - buka read_write.php
   - ketik sesuatu (klik load)
   - tambahkan sesuatu
   - klik save
   - liat perubahan di note.txt atau refresh page lalu load

3. upload file ke temp server
   - buka upload.php
   - upload filenya
   - liat filenya di C/xampp/tmp

4. upload ke dir server
   - buka upload_save.php
   - upload filenya
   - liat filenya di C/xampp/htdocs/ppwebb

5. buat database kayak di gambar
   database_name = 'ppwebb' atau bebas tapi ganti di tiap file nanti

6. upload file ke database
   - buka upload_database.php
   - upload file
   - liat file di database mysqlnya

7. list daftar file di database
   - buka download_database.php
   - klik download di salah satu
   - done
